# Food Ordering App
This is an online ordering system that works step by step
from choosing an item from the menu to the order summary

### This project includes:
- A menu where users can see the items and add to cart
- A cart appearing as soon as the user chooses an item
- A payment modal appearing when the "Complete Order" button is clicked
- A confirmation page with a summary of the order and the possiblity to start a new one
